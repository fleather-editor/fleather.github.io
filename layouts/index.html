All platforms supported

Fully functional on iOS, Android, macos, Windows, Linux and Web

Editor

Undo/redo functionality. Images can be embedded as block elements or inline element. Links can be embedded as inline elements. Markdown and system shortcuts are supported.

Formatting

Inline attributes such as bold, italic, strikethrough, underline and code. Line attributes can be assigned including direction, alignment, heading, indentation, ordered & unordered list. Block attributes including code, quote or embeds.

Conversion

Markdown codec enables encoding/decoding Parchment document. HTML codec is also available

Extensibility

Custom embeds can be used as inline or block embeds
